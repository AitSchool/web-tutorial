什么是因特网

因特网不等同于 web, 因特网范围更大、出现得更早、具有多样性。

想象一下纵横交错的道路是如何将整个世界连接起来的。从小街道到国道再到国际高速公路，你可以驾车到世界上的任何一个地方。值得注意的是，对于整个交通网来说并没有实际意义的中心。

因特网也和交通网类似的，它通过电缆来将世界上的计算机连接起来，并可在各个计算机之间传递数据信息。

最早在 1969 年，美国发明了可以将计算机连接起来的技术。如今，不仅仅是计算机，手提计算机、手机、TV等设备都可以通过因特网连接起来。

客户端和服务端

通常，一个连接是发生在 2 台计算机之间：

  • 一个具有数据信息（服务端）
  • 一台则想要另一台计算机中的信息（客户端）

客户端 可以是以下多种形式的软件:

  • 一个 Web 浏览器 (比如 火狐)
  • 一个邮件客户端 (比如 Outlook)
  • 一个聊天 app (比如微信)
  • 一个视频播放服务应用 (比如优酷)

上面的每一种客户端软件会从服务端(服务器)上检索需要的信息，服务端上会存储网页、邮件、聊天记录、电影等信息。客户端上通常是不会存储这些信息的而是当它需要的时候就到存有信息的服务器上去请求需要的信息。

服务器可以理解为一台始终连接到互联网上并用来传送数据的计算机。尽管任何的连接到互联网上的设备都可以既作为客户端，又可以作为服务端，但大多数我们所使用的设备一般都称为客户端，因为我们只用来接收数据而不是输出数据。

IP 地址

如同每一个房子都有一个独一的地址一样，IP 地址是每一台连接到网上的计算机在互联网上的唯一标识地址。IP 地址一般是四组数字的组合：91.198.174.192.

目前 IP 地址已经升级为 IPv6，地址也比之前的更为复杂。

域名

尽管互联网上的计算机可以通过 IP 地址来区分彼此，但是却对人类不友好，将一串数字作为地址来记录对人类来说是很痛苦的，所以就有了域名的概念。 域名创建于 1985 年，通过将文本形式的地址和IP地址绑定，这样用户就可以通过域名形式（与IP绑定的文本）或者 IP 来访问互联网上的计算机。

如 www.baidu.com 就是百度公司的域名，我们可以通过它或者百度公司计算机的 IP 地址来访问其服务。

域名由 3 个部分组成，按从左到右的顺序读。

  • 顶级域名(TLD): 通常为（.com, .org, .net）和国家级域名，如（.cn[中国], .us[美国]）,每个国家都会分配一个唯一的顶级的域名
  • 域名名称: 如 baidu, meituan 分别代表百度，美团的域名名称。可以为字符或数字
  • 子域名（可选）：一般为 www。

可以简单的将域名当做连接到互联网上的计算机的名字。

Protocols（协议）

将计算机连接到互联网上的目的是与其它连接的计算机进行通信，就如同人类用语言来进行交流一样，计算机通过 协议来交流通信。

Protocol Used for Created in
FTP File transfer 1971
SMTP Sending Emails 1971
IMAP Receiving Emails 1986
IRC Chat 1988
HTTP Browsing HTML documents (Webpages) 1989

URL（Uniform Resource Locator）

如下为百度搜索页地址 http://www.baidu.com/search.html , 可以分为三部分:

  • http:// 是使用的协议
  • www.baidu.com 是域名
  • /search.html 是页面的路径

通过 URL 可以知道从互联网的什么地方去找到页面以及使用什么样的协议去通信。

Internet 一张巨大的由许多计算机连接起来组成的网。

Protocol 如同语言一样，协议是计算机之间通信交流的语言，规则。

IP 地址 连接在互联网上的计算机的互联网地址。

Domain 域名是连接在互联网上的每一台计算机的独特名字，和 IP 地址不同的是，域名更容易阅读及记忆。
