什么是万维网

Web

我们已经知道计算机如何连接到互联网上以及通过不同的协议（语言）进行通信，发送邮件、文件、聊天等。

HTTP (超文本传输协议) HTTP 是一种用来在不同的计算机之间传送网页的协议。

万维网则是互联网上共享的网页的集合。我们可以通过 http 协议的形式在浏览器上打开一张网页。

网页

网页是一种通过 HTML 书写的在万维网上共享的文档，可以通过浏览器来访问。

网站

网站则是一个域名下的网页的集合。

  • Web http://
    • Website www.baidu.com
      • Webpage /introduction.html
      • Webpage /internet.html
      • Webpage /web.html

当我们在浏览器上输入一个页面地址时，浏览器就会去网上找到提供这个页面的服务器，然后将页面取回并展示出来。

Client Your computer Hi MarkSheet computer! I'd like the web.html file please
Server The MarkSheet computer Let me check if it's there...
Client Your computer Ok I'll wait
Server The MarkSheet computer Oh, here it is! Let me send that to you.
Client Your computer Got it. Thanks!

通过浏览器浏览网页的时候，并不会将页面下载到你的计算机上，而只会暂时的存储并浏览，这样可以保证每次访问页面的时候都会看到最新的内容。
